계통을 선택해 학습을 시작하세요

이 아틀라스가 하는 일

계통별로 용어 암기 → 구조 파악 → 임상 연계까지 하나의 흐름으로 학습합니다. 카드 암기는 간격반복(SRS) 방식이라 자주 틀리는 카드가 더 자주 등장합니다.

📋 오늘의 추천 학습 순서
1
⭐ 핵심 용어 암기 — 10개부터 시작
필터를 "핵심만"으로 설정하고 플래시카드를 먼저 돌리세요. 모두 Box 2 이상이 되면 다음 단계로.
2
🎮 암기 게임 — 어원 굳히기 & 속도 훈련
어근 빌더로 철자·의미를 익히고, 타이핑 스피드런으로 60초 안에 최대한 많이 맞히세요.
3
🗺 구조 시각화 — 위치 감각 잡기
각 신경·구조를 클릭해서 어디서 나와 어디로 가는지 흐름을 파악합니다. 연결된 카드·케이스로 바로 이동도 가능합니다.
4
🩺 임상 시나리오 — 실전 추론
증상을 먼저 읽고 스스로 진단해본 뒤 정답을 확인합니다. 케이스마다 자가진단 입력란이 있습니다.
5
📝 종합 테스트 — 최종 점검
용어+케이스를 섞어 무작위 출제. SRS 진행률에 영향 없는 별도 평가입니다.

해부학적 색 코딩(Netter 관례)을 그대로 사용합니다 — 신경은 노란색, 동맥은 빨간색, 정맥은 파란색, 근육은 갈색 계열로 표시합니다.

신경 (Nerve)
동맥 (Artery)
정맥 (Vein)
근육 (Muscle)

전체 진행률 (마스터한 카드 비율)

콘텐츠 공유 — 직접 추가한 카드/시나리오는 이 브라우저에만 저장됩니다. 파일로 내보내서 학생들에게 나눠주면, 학생들은 "가져오기"로 같은 카드를 받아볼 수 있습니다. (자동 동기화는 아니고, 파일 전달 방식입니다.)

용어 암기 (간격반복)

"기억났어요"를 누르면 다음 복습일이 늦춰지고, "다시 볼게요"를 누르면 곧 다시 나옵니다.

암기 게임

조각을 순서대로 클릭해서 실제 용어를 조립하세요. 정답 개수가 늘어날수록 헷갈리는 조각(오답 타일)이 많아집니다.

60초 안에 설명을 읽고 용어를 최대한 많이 타이핑해 맞히세요. 오탈자는 어느 정도 허용됩니다.

구조 시각화

각 단계를 클릭하면 기시, 지배 영역, 손상 시 증상이 오른쪽에 표시됩니다. 도식은 이해를 돕는 단순화된 그림이라, 아래 실제 도판과 함께 보세요.

왼쪽에서 구조를 클릭해 보세요.

임상 시나리오

증상을 먼저 읽고 손상 부위를 추론해 보세요. 클릭하면 정답과 해설이 열립니다.

종합 테스트

용어와 임상 케이스를 섞어 무작위로 출제합니다. SRS 진행률에는 영향을 주지 않는 별도 평가 모드입니다.

🖨 시험지 생성

설정을 고른 뒤 미리보기를 확인하고 인쇄하세요. 학생용(빈칸)과 교사용 정답지가 함께 생성됩니다.
